# Even More Information

All potential candidates deserve equal and fair access to information about the
FedRAMP Cybersecurity Service, so we've enabled discussions on the
[FedRAMP/join](https://github.com/FedRAMP/join/discussions) GitHub repository.

We strongly recommend watching this repository for updates.

## Questions via Email

If you would prefer not to use GitHub to ask questions (for many perfectly
good reasons in a public discussion forum about hiring), you may instead
email the FedRAMP Director at **pete@fedramp.gov**.

Unless your question is very specific and not applicable to the public, we'll
post the question and answer on your behalf to ensure everyone gets the
same information.
